# Filho do Céu

[![License](https://img.shields.io/github/license/ederchristian/filhodoceu.com.br?color=%2300d59f)](https://github.com/ederchristian/filhodoceu.com.br/blob/main/LICENSE)

> Site com a listagem de 10 músicas de Padre Fábio de Melo utilizando a API do Spotify

## Motivação

Padre Fábio de melo é uma pessoa que me identifico desde criança. Ele é uma pessoa importante não só pra mim, mas pra milhões de pessoas no mundo.

Criei esse _side project_ com o objetivo de divulgar um pouco mais do ministério dele. Com este projeto, divulgo algumas músicas que me marcaram.

A ideia é criar outros projetos nesse estilo pra divulgar mais ainda. Assim como o trabalho que ele produz me faz bem, acredito que pode também fazer bem a mais pessoas ainda.

## Tecnologias utilizadas

Esse _side project_ é um projeto bem simples. Foram utilizados:

- HTML
- CSS
- JavaScript
- Biblioteca [spotify-web-api-node](https://github.com/thelinmichael/spotify-web-api-node) para trazer uma playlist de músicas do Spotify no formato JSON

## Instalação

Basta abrir o arquivo `index.html` no seu navegador.

Você pode usar um servidor de sua preferência. Se você usa VS Code, recomendo utilizar a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer). Basta clicar no botão "Go Live", após a instalação. Ele abrirá na porta http://127.0.0.1:5500/
